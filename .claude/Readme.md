TomaReco（トマレコ）仕様書

🧩 アプリ概要

TomaRecoは、日々の学習をシンプルに記録し、カレンダー形式で可視化する習慣化アプリです。
ユーザーは学習内容を記録し、継続日数（streak）を確認することでモチベーションを維持できます。

⸻

🎯 目的
	•	学習を継続するための仕組みを提供する
	•	学習履歴を可視化する
	•	シンプルで使いやすいUIでストレスなく記録できるようにする

⸻

👤 想定ユーザー
	•	勉強を習慣化したい人
	•	記録をシンプルに残したい人
	•	アプリに時間を取られたくない人

⸻

⚙️ 現在の機能

📅 カレンダー機能
	•	月ごとのカレンダー表示
	•	学習した日にマーク表示
	•	日付タップで詳細表示

📝 記録機能
	•	日付ごとに学習内容を保存
	•	タイトル + ジャンルで記録
	•	ローカル保存（AsyncStorage）

🏷 ジャンル機能
	•	ジャンルの追加
	•	ジャンルごとの色分け
	•	ジャンル削除

🔥 継続日数（streak）
	•	連続して学習した日数を表示

📆 日付操作
	•	月移動（スワイプ・ボタン）
	•	DatePickerによる日付変更

⸻

🧠 データ構造

Record

type Record = {
  year: number;
  month: number;
  day: number;
  title: string;
  genreId: string;
};

Genre

type Genre = {
  id: string;
  name: string;
  color: string;
};


⸻

💾 データ保存
	•	AsyncStorageを使用
	•	すべて端末内に保存
	•	外部サーバー通信なし

	•	記録のハードルを極限まで下げる
